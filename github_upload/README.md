# GitHub 上传专用目录（图片）

你要上传到 GitHub 的图片已经整理在：

- `github_upload/images/`

包含文件：

- `douyin_explosive_style_cover.png`
- `douyin_checklist_board_cover.png`
- `douyin_explosive_style_cover.svg`
- `douyin_checklist_board_cover.svg`

上传到 GitHub 后，你可以用下面路径直接访问：

- `https://github.com/<your-account>/test-codex/tree/main/github_upload/images`

单图直达（示例）：

- `https://github.com/<your-account>/test-codex/blob/main/github_upload/images/douyin_explosive_style_cover.png`
- `https://github.com/<your-account>/test-codex/blob/main/github_upload/images/douyin_checklist_board_cover.png`

Raw 下载直链格式：

- `https://raw.githubusercontent.com/<your-account>/test-codex/main/github_upload/images/douyin_explosive_style_cover.png`
- `https://raw.githubusercontent.com/<your-account>/test-codex/main/github_upload/images/douyin_checklist_board_cover.png`
