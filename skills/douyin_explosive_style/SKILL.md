# Skill: Douyin High-Traffic Image-Post Style System

## Purpose
Provide a reusable style system for high-traffic Douyin image-post layouts, grounded in public trend observations and adapted for 4-5 page high-density conversion content.

## Style Pack (可组合)

### 1) Red-Black Alert（高冲突警示风）
- Use case: 诊断类、避坑类、复盘类
- Cover formula: 人群 + 痛点 + 结果预期
- Visual rules:
  - 背景深色（黑/深灰）
  - 核心词红色高亮（不超过2处）
  - 主标题2行内，副标题1行

### 2) Checklist Board（清单白板风）
- Use case: 教程、步骤、SOP
- Visual rules:
  - 白底 + 黑字 + 单色强调（蓝/红）
  - 每页 3~5 条要点
  - 错误与修复使用左右对照

### 3) Before/After Split（前后对比风）
- Use case: 优化前后、投放复盘
- Visual rules:
  - 画面一分为二（Before / After）
  - 数据只放 1~2 个关键指标
  - 禁止堆砌超过3个数字

### 4) Data Card Minimal（数据卡片极简风）
- Use case: 专业背书、行业报告摘要
- Visual rules:
  - 每页一个结论 + 一张数据卡
  - 大数字 + 小注释
  - 保留30%留白，防止信息拥堵

## 4-5 Page Layout Blueprint
- P1: 封面钩子（人群+痛点+结果）
- P2: 错误1 + 修复1
- P3: 错误2 + 修复2
- P4: 错误3 + 修复3
- P5: 单一 CTA + 交付物 + 时效（可选）

## Typography
- 标题：粗体黑体（建议 80-120 px）
- 正文：中黑（建议 42-56 px）
- 行间距：1.25~1.35
- 单页文字量：45~90 汉字

## Safety & Compliance
- 禁止绝对化承诺（如“保证收益”）
- 避免过度营销诱导
- CTA 单一动作，减少误导
