# Skill: Douyin 4-5 Page High-Density Error/Fix Copy

## 目的
将图文内容压缩为 4~5 页，并在单页内同时完成“错误诊断 + 修复动作”，提高信息密度与转化效率。

## 适用场景
- 本地商家获客图文
- 代运营交付模板
- 发布前快速改稿

## 执行规范
1. 总页数固定 4~5 页。
2. 第 1 页只做定位：人群、痛点、结果预期。
3. 第 2 页起每页必须包含：
   - 错误：用户常见错误或误区
   - 修复：一步可执行修复动作
4. 每页正文 45~90 字，必须有可执行信息。
5. 末页只留 1 个 CTA（评论关键词或私信关键词）。

## 页面模板
- P1：封面钩子（人群+痛点+结果）
- P2：错误1 + 修复1
- P3：错误2 + 修复2
- P4：错误3 + 修复3
- P5（可选）：CTA + 交付物承诺

## 质检清单
- 是否出现空泛口号？
- 是否每页都有新信息增量？
- 是否错误和修复在同一页？
- CTA 是否只有一个动作？
