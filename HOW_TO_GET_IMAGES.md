# 如何在 GitHub 获取素材（本次 PR 为 SVG + Markdown）

> 你反馈“创建拉取请求不支持二进制文件”，本次已改为仅提交 SVG 和 Markdown。

## 下载 SVG 源文件（可编辑）
1. 打开仓库主页。
2. 进入 `output/source/`。
3. 点击任意 `*.svg` 文件。
4. 在文件页面点击 **Download raw file**。

## 备用 SVG 目录
- `github_upload/images/`（仅保留 SVG）
- `assets/examples/`（仅保留 SVG + README）
