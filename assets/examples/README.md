# Style Examples Preview（升级版）

你提到“风格、排版、字体太丑”，这一版已做了可直接复用的视觉升级：

- 字体：统一中文优先字体栈 `Noto Sans SC / PingFang SC / Microsoft YaHei`。
- 排版：统一标题区、内容卡片区、CTA区三级结构。
- 信息密度：每张图都采用“错误 + 修复”同页表达。
- 风格：覆盖 5 种可落地模板（警示、清单、故事、数据卡片、反差对比）。

## 5 个示例（SVG）
- `output/source/001_alert_red_black.svg`（黑红警示风）
- `output/source/002_checklist_blue_white.svg`（蓝白清单风）
- `output/source/003_orange_story.svg`（橙色故事风）
- `output/source/004_green_data.svg`（绿色数据卡片风）
- `output/source/005_purple_contrast.svg`（紫色反差对比风）

## 使用建议
1. 先选行业（教培/医美/家装/本地生活）。
2. 再选视觉风格（上面 5 套之一）。
3. 最后替换标题、痛点和 CTA 关键词。
